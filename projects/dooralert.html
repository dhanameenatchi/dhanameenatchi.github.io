Aim to Develop this Project

  • This project aims to develop a door alert system using Python, OpenCV, and face recognition modules.

  • The system will use a camera to capture images of individuals attempting to enter a designated area, such as a home or office.

    • Architecture of Project

    Input Requirements

  • Input requirements at base include photo and mobile number from the user.
  • These photos and mobile number are passed as data to our system.
  • Apart from this the basic requirements include stable internet connection.
  • A camera with high pixel quality.

    • Output Requirements

  • A person should not wear mask and he/she should stand at a particular distance.

    • How it Works?

    • web cam is used for capturing the images of students or employees.

    • The faces in the captured images are detected and compared with the images in database and the door will be alerted.

    • This system deals with door alert system using opencv and face recognition packages from which we can capture the images , detect the face on the image and extract the face coordinates.

    • The value extracted from the image is compared with database of images and if a match is found face detected for the respective person with date and time using datetime module also a text message is send to person using Twilio or Discord.
